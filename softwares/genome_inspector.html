About Genome Inspector

Your comprehensive tool for DNA sequence analysis

Welcome to Genome Inspector 1.0

Genome Inspector 1.0 is an advanced graphical user interface (GUI) application based on BioPython specifically designed to facilitate comprehensive DNA sequence analyses. Whether you are a researcher, student, or simply someone with a keen interest in molecular biology, or bioinformatics who wants to analyse the genome, Genome Inspector 1.0 offers a user-friendly and intuitive platform to meet your needs.

User-Friendly Design

This application is built with ease of use in mind. The interface, crafted using the Tkinter library, ensures a smooth and interactive experience. You don't need to be a seasoned bioinformatician to navigate and utilize the powerful features of Genome Inspector 1.0. With clear menus, toolbars, and visual aids, you can quickly get up to speed and start analyzing your DNA sequences efficiently. You can also save your results in text file format for future use.

Installation Guide

Get started in minutes

Windows

  1. Download the Windows installer (.exe) file
  2. Double-click on the downloaded file to run the program
  3. That's it! Start analyzing your DNA sequences

Linux

  1. Ensure Python is installed. Install BioPython and Tkinter if needed:
    pip install biopython
    pip install tkinter
  2. Extract the TAR file:
    tar -zxvf Genome_Inspector.tar.gz
  3. Navigate to directory and run setup:
    cd Genome_Inspector
    chmod +x setup
    ./setup
  4. Run the program:
    geneinspect
